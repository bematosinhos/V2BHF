## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

1. Clone the repository

```bash
git clone https://github.com/m4n3z40/sbc-cursor-starter-kit.git
cd sbc-cursor-starter-kit
```

2. Install dependencies

```bash
npm install
# or
pnpm install
```

3. Start development server

```bash
npm run dev
# or
pnpm dev
```
